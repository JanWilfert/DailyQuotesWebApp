# DailyQuotes

The code for the Android app is in a seperate repository [here](https://github.com/Emil-96/dailyquotes_android).

## change directory

```sh
cd ./WebApp
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
